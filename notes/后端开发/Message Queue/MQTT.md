运行的时候走到bin目录外运行 emqx start 不要进去bin目录

```bash
wget https://www.emqx.com/zh/downloads/broker/5.0.4/emqx-5.0.4-el7-amd64.tar.gz

mkdir -p emqx && tar -zxvf emqx-5.0.4-el7-amd64.tar.gz -C emqx

./emqx/bin/emqx start
```
